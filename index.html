機械学習03-データ前処理

投稿日 2020-04-19 | In AI | Visitors

モデル学習するためのデータを事前に処理しないといけないです。
今回はsklearnのメソッドでデータ前処理を勉強します。

1、データの正規化（数据归一化　Normalization）
最小値0、最大値1に正規化するにはMinMaxScalerクラスを使います。MinMaxScalerには重要な引数feature_rangeがあります。正規化したい範囲と定義されて、デフォルトは「０、１」です。

from sklearn.preprocessing import MinMaxScaler
data = [[-1, 2], [-0.5, 6], [0, 10], [1, 18]]
s = MinMaxScaler()
result= s.fit_transform(data)
array([[0. , 0. ], [0.25, 0.25],[0.5 , 0.5 ],[1. , 1. ]])

scaler = MinMaxScaler(feature_range=[5,10])

元の配列に戻るなら、s.inverse_transform(result)

２、データの標準化（数据标准化　Standardization　均值为0，方差为1的正态分）
平均0、分散1に標準化するにはStandardScalerクラスまたはscale()関数を使います。

　今回はランダムフォレストについて勉強したいと思います。

機械学習02 アンサンブル学習（集団学習）_ランダムフォレスト

複数の学習器を利用して高い精度を出すことを実現するのはアンサンブル学習ですね。

分類は以下の３つになります。
●　Bagging(バギング)
●　Boosting(ブースティング)
●　Stacking(スタッキング)

Baggingは独立の学習機で作られます。その中で、弱学習器として決定木を用い、バギングしたものが、ランダムフォレストになります。決定木の評価が高くなるとランダムフォレストの評価も高くなります。

新しいパラメータ：
n_estimators：整数を指定．デフォルトの値は100。バギングに用いる決定木の個数を指定。

機械学習01-決定木の回帰木

投稿日 2020-04-14 | In AI | Visitors

DecisionTreeRegressor

class sklearn.tree.DecisionTreeRegressor (criterion=’mse’, splitter=’best’, max_depth=None, min_samples_split=2, min_samples_leaf=1, min_weight_fraction_leaf=0.0, max_features=None, random_state=None, max_leaf_nodes=None, min_impurity_decrease=0.0, min_impurity_split=None, presort=False)

criterion
mse:使用均方误差mean squared error(MSE)，父节点和叶子节点之间的均方误差的差额将被用来作为 特征选择的标准
回归树的接口score返回的是R平方，并不是MSE

交差検証（Cross-validation）
　汎化精度を高めることが目的。交差検証では，データセットを kk 個に分割し，モデルの訓練と評価を kk 回行います。
得られた kk 個の評価値の平均をとった値を最終的なモデルのスコアとして扱います。

1.データを kk 個のブロックに分ける．これを分割（fold）という．
2.最初の分割1 を test set，残りの分割2~5 を training set とし，モデルの学習と評価を行う．
3.分割2 を test set，残りの分割1, 3~5 を training set として，モデルの学習と評価を行う．
4.この過程を，分割3, 4, 5 を test set として繰り返す．
5.得られた kk 個の精度の平均値をモデルの評価値とする．
