オブジェクト移動と回転方法のまとめ

投稿日 2017-11-20 | In unity3d | Visitors

大変お久しぶり。
この前、Unity認定試験とか準備のため時間かかりました。無事合格しましたが、点数はまあまあです。。。やはり基礎は大事ですね。
まあ、今日はUnityで一番よく使われ、基本的なオブジェクトの移動と回転につて、まとめたいと思います！
シーンの中ですべてのオブジェクトが持っているコンポーネントはTransformですね。先ずは移動です。方法は大体五つがあります：
1.Transformを使って移動する
2.Rigidbodyを使って移動する
3.CharactorControllerを使った移動
4.Vector３を使って移動する
5.NavMeshを使って移動する
`

Transformを使って移動する
1.transform.positionを書き換える
オブジェクトのトランスフォーム座標情報を書き換えるため、他のオブジェクトトの衝突判定をすり抜けやすいです。ワープ現象。

transform.positionはオブジェクトのワールド座標であり、transform.forwardは自身ローカル座標は相対ワールド座標での前方向ですね。
2.transform.Translate
public void Translate(Vector3 translation, Space relativeTo = Space.Self);
移動は Transform のローカル軸/ワールド軸で移動する。

Rigidbodyを使って移動する
Rigidbodyコンポーネントをアタッチしておく必要があります。
Rigidbody.AddForceを使って移動する

Rigidbody.velocityを書き換えて移動する

Rigidbody の interpolation （補間）設定にしたがって、Rigidbody.MovePosition も使えるようになります。

CharactorControllerをアタッチします。重力が無効です。

Vectorを使った移動方法
MoveTowards
public static Vector3 MoveTowards(Vector3 current,
Vector3 target, float maxDistanceDelta);
現在の位置 current から target に向けて移動します。

Vector3.Lerp / Vector3.Slerp
指定した位置まで滑らかに移動させてくれる

回転
回転方法の分類
参考：Unityでオブジェクトを回転させる方法まとめ

指定した角度までなめらかに回転させたい時に便利な方法
Mathf.LerpAngleを使って回転する
Quaternion.LookRotationを使って回転する
Quaternion.RotateTowardsを使って回転する
Vector3.RotateTowardsを使って回転する
Quaternion.Slerpを使って回転する
指定した角度まで一気に回転させたい時に便利な方法
Quaternion.AngleAxisを使って回転する
Quaternion.Eulerを使って回転する
Quaternion.eulerAnglesを使って回転する
transform.eulerAnglesを使って回転する

Transformのオイラー角(Vector3 eulerAngles)を書き換える。
transform.eulerAngles

Transform.Rotateという方法はずっと回転ほしい場合使う
public void Rotate(Vector3 eulerAngles, Space relativeTo = Space.Self);

public static Quaternion LookRotation(Vector3 forward, Vector3 upwards = Vector3.up);
Quaternion.Slerpと併用して、なめらかに敵の方を向かせる処理などに使われる例が多い。

Quaternion.Eulerを使います。
public static Quaternion Euler(float x, float y, float z);

今回終わりです。また次回！
